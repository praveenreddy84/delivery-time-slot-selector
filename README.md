# delivery-time-slot-selector
A Chrome extension for Amazon whole foods delivery time slot auto selectors <br> 


  Steps to Setup<br>
  Step 1: Download the code <br>
  Step 2: Go to chrome extensions chrome://extensions<br>
  Step 3: Enable Developer Mode on Extenstions page<br>
  Step 4: Click on loadupacked button and select the extension folder downloaded from this repository.<br>
  Step 5: Now login to your amazon.com add some items to your wholefoods cart, and once you are this page "https://www.amazon.com/gp/buy/spc/handlers/display.html?hasWorkingJavascript=1" sit back and relax. To view the progress open your console and look for the activity.<br>

Enjoy!!
